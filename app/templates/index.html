What is Thousandblurbs?

Thousandblurbs is a web application starter kit built for the cloud using Google App Engine, Flask, and Bootsmooth.

What kind of web applications can you build with Thousandblurbs?

  1. Small, fast websites and landing pages with free hosting quota*
  2. Form collection with administrative interface
  3. Backend micro services and data collection API for mobile apps
  4. Complete, full-stack web applications
* Google App Engine offers a free usage tier (as of Jan 2017)

Features include:

  • Sass-based stylesheet built with Bootsmooth
  • Customizable Jinja2 templates for web pages and admin dashboard
  • Form capture service and maintenance using Google Cloud Datastore
